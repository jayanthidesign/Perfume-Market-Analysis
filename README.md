# Perfume Market Analysis

# Project Overview:

This project analyzes perfume sales data by combining two separate datasets: men’s
perfumes and women’s perfumes. The datasets were merged into a single unified
dataset to enable comprehensive analysis across all product categories.
By integrating both datasets, the analysis provides a holistic view of sales trends,
customer preferences, and product performance across different segments.

# Tools & Technologies:

Language: Python (Pandas, NumPy, Seaborn) 

# Data Cleaning & Preparation:

Cleaned both datasets (removed duplicates, handled missing values,
standardized columns)

Ensured consistent column names and data types.

Added the category column to distinguish between Men/Women.

Merged both datasets into a single dataset using: Python.

# Key Insights:

New Jersey (~0.64M) and Texas (~0.48M) dominate sales, indicating strong
market demand in these states.

Cities like San Francisco and Portland have higher stock availability, while others
have limited stock → possible inventory imbalance.

Categories like skin moisturizer and perfume oil contribute very little revenue →
low demand segments.

 # Recommendations:

By combining the men’s and women’s perfume datasets, a complete view of
overall sales performance was achieved. The analysis highlights key trends in customer
preferences, pricing, and regional demand, helping identify top-performing products
and areas for improvement. These insights support better decision-making in inventory
management, pricing strategies, and targeted marketing to enhance overall business
performance.
